## Install

Copy this folder to:

```text
~/.codex/pets/ladybug-dev/
```

Then open Codex App, go to `Settings > Appearance > Pets`, refresh custom pets, select `Ladybug Dev`, and type `/pet`.

## Brief

Ladybug Dev is a tiny coding ladybug with a red shell, terminal-marked spots, expressive eyes, tiny legs, and a simple laptop.

## States

- Idle: codes quietly at the laptop.
- Running: scurries with the laptop open.
- Waiting: watches for input with bright focus.
- Review: inspects the laptop with tiny developer seriousness.
